We want to get count of  markets which __visible=1__ and symbols which theirs name end with __DAI__

```shell
curl -s  "https://api-enifex-dev.ernyka.com/market/all-symbol-thumb"  | jq '.[]'  | jq 'if .visible == 1 then .symbol else 0 end' | egrep '*/DAI' | wc -l
```

## 1- jq '.[]'
Because response is numerical we want the body of first key (numerical)

## 2- jq 'if .visible == 1 then .symbol else 0 end'
if visible == 1 then return values of which theirs keys are __symbol__ else return __0__
